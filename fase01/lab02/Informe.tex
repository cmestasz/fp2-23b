\section{Tarea}
\begin{itemize}
	\item \textbf{Actividad 1:} En este ejercicio se le solicita a usted implementar el juego del ahorcado utilizando el código parcial que
	      se le entrega. Deberá considerar que:
	      \begin{itemize}
		      \item El juego valida el ingreso de letras solamente. En caso el usuario ingrese un carácter equivocado
		            le dará el mensaje de error y volverá a solicitar el ingreso.
		      \item El juego supone que el usuario no ingresa una letra ingresada previamente.
		      \item El método ingreseLetra() debe ser modificado para incluir las consideraciones de validación.
		      \item Puede crear métodos adicionales.
	      \end{itemize}
\end{itemize}

\section{Código desarrollado}
\lstinputlisting[language=Java, caption={Ahorcado.java},numbers=left,]{Ahorcado.java}
\begin{itemize}
	\item Se inicializan variables con los posibles estados del juego, posibles palabras secretas y un contador de turnos.
	\item getPalabraSecreta() regresa una palabra escogida al azar del arreglo palabras.
	\item mostrarBlancos() imprime la cantidad de letras que se tienen que descubrir.
	\item El juego inicia y continua hasta que el contador de errores sea 6 o la palabra este completa, verificado por el método verificarPalabraCompleta().
	\item En cada turno, ingreseLetra() recibe una letra del usuario, la que es verificada por el método esLetraValida(), letraEnPalabraSecreta() verifica si la letra es parte de la palabra secreta,
	      y si lo es, mostrarBlancosActualizados() actualiza e imprime las letras encontradas y las que faltan encontrar.
	\item Si la letra no está en la palabra secreta, el contador de errores aumenta.
	\item Una vez el ciclo ha terminado, verificarPalabraCompleta() devuelve si la palabra esta completa y se imprime el resultado al usuario.
	\item Además se imprime la cantidad de turnos totales, los que aumentaban por cada iteración en el bucle anterior.
\end{itemize}
